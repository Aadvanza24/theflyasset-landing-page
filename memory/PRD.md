# THE FLY ASSET - Portfolio Website

## Problem Statement
Static portfolio website for "THE FLY ASSET," a trading and market research firm based in Kolkata, India.

## Architecture
- **Frontend only** (React + Tailwind CSS, client-side routing via react-router-dom)
- Minimal FastAPI backend with `/health` endpoint for deployment health checks
- Build tool: CRACO (Create React App Configuration Override)

## Pages
Home, About, Capabilities, Trading Process, Insights, Contact, Disclaimer, Risk Disclosure, Privacy Policy, Terms of Use

## Design
Light theme: white background, navy blue (#0a1628) text, gold (#D4AF37) accents. Header background: #0c102e with white menu text.
