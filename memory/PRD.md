# THE FLY ASSET - Portfolio Website

## Problem Statement
Static portfolio website for "THE FLY ASSET," a proprietary trading and market research firm based in Kolkata, India.

## Architecture
- **Frontend only** (React + Tailwind CSS, client-side routing via react-router-dom)
- No backend/database required — pure static site deployable via `yarn build`
- Build tool: CRACO (Create React App Configuration Override)

## Pages
Home, About, Capabilities, Trading Process, Insights, Contact, Disclaimer, Risk Disclosure, Privacy Policy, Terms of Use

## Design
Light theme: white background, navy blue (#0a1628) text, gold (#D4AF37) accents, navy header. Professional Indian-focused imagery.

## What's Been Implemented
- Full multi-page static React website with all 10 pages
- Navy blue header, light theme with gold accents
- Professional Indian-focused imagery throughout
- Footer with "Ads Idea Tech" credit link and educational disclaimer text
